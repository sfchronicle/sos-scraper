# SOS-scraper
Python scraper created using Jupyter to pull businesses data from the Secretary of State.

### Requirements
- Python 2.7.x (or 3..?)
- Jupyter (`pip install jupyter`)

### Installation
```bash
$ git clone git@github.com:sfchronicle/sos-scraper.git scraper
$ cd scraper
$ mkvirtualenv scraper
$ pip install jupyter
$ jupyter notebook
```
